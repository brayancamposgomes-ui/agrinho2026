# 🥔 Batata Sustentável

Um site educativo que explora a batata como alimento essencial e também como fonte alternativa de energia.  
O projeto combina **nutrição, sustentabilidade e ciência** em uma experiência interativa e visual.

## 📖 Seções do Site

- [História](ca://s?q=Detalhar_historia_batata): origem nos Andes e papel na segurança alimentar
- [Alimentação](ca://s?q=Batata_na_alimentacao): valor nutricional e versatilidade culinária
- [Energia](ca://s?q=Batata_como_energia): experimentos científicos com batata como bateria
- [Curiosidades](ca://s?q=Curiosidades_sobre_batata): cultivo no espaço e fatos interessantes
- [Galeria](ca://s?q=Galeria_de_imagens_batata): imagens ilustrativas
- [Contato](ca://s?q=Contato_no_site): seção para interação com visitantes

## 🎨 Estilo e Layout

- **Fonte:** Verdana, sans-serif  
- **Cores principais:**  
  - Cabeçalho: gradiente marrom (#8B4513 → #A0522D)  
  - Navegação: bege (#deb887)  
  - Fundo: creme (#fffaf0)  
- **Design responsivo:** seções centralizadas com largura máxima de 900px
